查看 API 文档

lkmio/lkm

高性能流媒体服务器，支持 RTMP, RTSP, WebRTC, HTTP-FLV 等多种协议，利用 Go 语言的高并发特性，提供稳定可靠的流媒体处理能力。

lkmio/gb-cms

GB28181 信令服务核心实现，全面支持国标28181协议（注册、目录、预览等），并独家支持 JT1078 到 GB28181 的协议转换，完美适配 LiveGBS 前端。

lkmio/avformat

专注于音视频容器格式处理的库，提供高效的解复用（Demuxing）和复用（Muxing）功能，是流媒体服务的底层基石。

API 友好

提供清晰的 RESTful API，方便集成和二次开发，适配 LiveGBS 前端。
