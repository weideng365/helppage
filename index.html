高性能 Go 实现

利用 Go 语言的高并发特性，提供稳定可靠的信令处理能力。

GB28181 标准支持

全面支持国标28181协议，包括设备注册、目录查询、视频预览等核心功能。

JT1078 互通

独有的 JT1078 到 GB28181 转换功能，打通部标与国标视频流。

API 友好

提供清晰的 RESTful API，方便集成和二次开发，适配 LiveGBS 前端。
